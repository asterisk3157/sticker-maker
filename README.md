# Sticker Maker

ブラウザだけで完結する、2L判ステッカー作成ツール。

PNG画像を読み込んで配置・拡大縮小・回転で組み合わせ、印刷向けのJPEG画像として書き出します。コンビニ等のネットワーク印刷サービスで2L判印刷する用途を想定しています。

## 特徴

- ブラウザ上で完結（**画像はサーバーに送信されません**）
- 自動保存（ブラウザの localStorage）
- 実寸表示（クレジットカードでキャリブレーション）
- ネットワーク印刷向けの安全マージン表示（外側は灰色のガイド領域）
- 縦/横切替、Undo/Redo、コピー&ペースト
- マウス中心の操作、矢印キーで微調整

## 使い方

1. 「PNGインポート」または右サイドバーへドラッグして画像を読み込み
2. ライブラリのサムネをキャンバスにドラッグ&ドロップ
3. 配置・拡大縮小・回転で並べる
4. 「JPEGエクスポート」で2L判サイズのJPEG画像を保存
5. お好きなネットワーク印刷サービスにアップロードして印刷

## キーボードショートカット

| キー | 動作 |
|------|------|
| `Cmd/Ctrl + Z` | 元に戻す (Undo) |
| `Cmd/Ctrl + Shift + Z` / `Cmd/Ctrl + Y` | やり直す (Redo) |
| `Cmd/Ctrl + C` | コピー |
| `Cmd/Ctrl + V` | 貼り付け |
| `Cmd/Ctrl + D` | 複製 |
| `Delete` / `Backspace` | 選択中ステッカーを削除 |
| 矢印キー | 微調整移動（`Shift` で20px単位） |
| `[` / `]` | 重なり順を1段下/上 |

## 開発・実行

ビルド不要。`index.html` をブラウザで開くだけで動作します。
